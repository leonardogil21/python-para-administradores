# Lógica de Programação para Administração

Este repositório contém exercícios práticos de **lógica de programação em Python aplicados à área de administração e negócios**.

O material foi desenvolvido como parte das aulas de programação, com foco em resolver problemas comuns de gestão utilizando conceitos básicos de programação.

## Objetivo

Demonstrar como conceitos fundamentais de programação podem ser aplicados em cenários administrativos, como:

- análise de vendas
- classificação de desempenho
- cálculo de médias
- controle de estoque
- análise de margem de lucro

## Conteúdo do Projeto

O notebook inclui exercícios sobre:

### 1. Estruturas Condicionais
Uso de `if`, `elif` e `else` para tomada de decisão.

Exemplos:
- Classificação do desempenho de vendas
- Análise de margem de lucro

### 2. Estruturas de Repetição
Uso de loops como `for` para processar listas de dados.

Exemplos:
- Cálculo da média de vendas semanais
- Contagem de devoluções de produtos

### 3. Manipulação básica de dados
Uso de bibliotecas como:

- pandas
- seaborn
- matplotlib

para análise e visualização simples de dados.

## Tecnologias Utilizadas

- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn
